没必要死记

不过也有方法：初始化两个指针，然后进行指针移动类似swap的环环相扣，被赋值的引用（左operand）必定在上一行作为赋值医用（右operand）

```java
public ListNode reverseList(ListNode head) {
          ListNode prev = null;
        ListNode curr = head;
        while (curr != null) {
            ListNode temp= prev;
            prev = curr;
            curr = curr.next;
            prev.next = temp;
            
        }
        return prev;

    }

```
